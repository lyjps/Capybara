# 聚餐执行清单（管家内部参考）

## 聚餐前 1-2 周
- [ ] 确定时间、地点、人数
- [ ] 预订餐厅（包间/大桌）
- [ ] 发送邀请并收集确认

## 聚餐前 3 天
- [ ] 再次确认人数，通知餐厅
- [ ] 确认特殊饮食需求（素食/过敏）
- [ ] 查看天气预报，准备出行建议
- [ ] 如有庆生：安排蛋糕/布置

## 聚餐前 1 天
- [ ] 提醒所有参与者时间地点
- [ ] 确认代驾安排（如有饮酒计划）
- [ ] 准备好场景装饰/小礼物（如需要）

## 聚餐当天
- [ ] 提前到达餐厅确认包间就绪
- [ ] 告知大家集合地点和停车信息
- [ ] 提醒拍集体照
- [ ] 结账方式确认（AA/谁请客）

## 场景特殊提醒

### 有老人/小孩
- 确认餐厅有无儿童座椅
- 选噪音相对小的包间
- 菜品注意口味清淡、易嚼

### 商务宴请
- 提前了解对方饮食禁忌（宗教/过敏）
- 确认包间隔音和设备（投影/WiFi）
- 备好名片
- 不强劝酒，准备无酒精饮料选项

### 庆生聚会
- 安排蛋糕（提前预订或餐厅协助）
- 准备 BGM 和生日歌
- 如果要惊喜，安排好"剧本"
- 记得拍视频/照片

### 有饮酒计划
- 提前预约代驾（旺季/节假日需提前1-2天）
- 告知大家饮酒后代驾信息
- 备好解酒药/热水

## 常见坑点提醒（管家主动说）
1. 节假日餐厅爆满 → 提前至少3-5天预订
2. 包间有最低消费 → 确认清楚，别到时候超出预算
3. 停车位少 → 建议地铁到的和开车的分别规划
4. 菜上太慢 → 预订时说明是聚餐，请餐厅优先上菜
5. 人多容易拖时间 → 设定"X点集合，X点开吃"并提醒
