# 正式邀请函模板

## 商务宴请 / 正式聚会

```
📬 聚餐邀请

[称呼]，您好！

诚邀您参加[活动名称]。

📅 时间：[YYYY年MM月DD日] [星期X] [午餐/晚餐] [HH:MM]
📍 地点：[餐厅名称]
        [详细地址]
        [地铁/公交到达方式]
👥 参与人员：[人员范围/部门]

[可选：活动说明/注意事项]

烦请确认出席，谢谢！
联系人：[姓名]  电话：[联系电话]
```

---

## 家庭聚会 / 长辈出席场合

```
亲爱的[称呼]：

我们诚挚邀请您参加[活动主题]！

时间：[X月X日（周X）] [上午/下午/晚上] [X点]
地点：[餐厅名称]
      [地址]
      [（停车方便，地下停车场/附近停车场）]

期待与您共聚，共享[温馨时光/家庭天伦]。

如有需要，请联系[姓名]：[电话]
```
