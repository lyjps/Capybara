# 找优惠 — 领取优惠券

## 工具：`get_coupon`

无需任何参数，直接调用即可。工具服务器会帮用户领取当前可用的优惠券。

## 调用方式

```bash
curl -s -X POST http://localhost:3456/tool \
  -H "Content-Type: application/json" \
  -d '{"tool": "get_coupon", "params": {}}'
```

## 使用场景

- 用户问"有没有优惠券"、"能省钱吗"、"有折扣吗"
- 用户下单/预订前主动提醒可以领券
- 搜索到美团商家后，询问是否要领券

## 注意事项

- 该工具不需要任何参数
- 调用后根据返回结果告知用户领取情况
- 如返回无可用优惠券，告知用户当前暂无可用券
